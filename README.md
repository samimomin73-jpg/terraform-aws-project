# 🚀 AWS Infrastructure using Terraform

This project provisions AWS infrastructure using Terraform using a modular approach.

## 📌 Project Structure

cloud-inst-project/
- main.tf
- provider.tf
- variables.tf
- modules/

## ⚙️ Services Used

- VPC
- EC2
- S3
- ALB
- Lambda
- DynamoDB
- Route53
- IAM
- KMS
- EFS
- EBS
- SNS

## ▶️ How to Run

terraform init  
terraform plan  
terraform apply  

## ⚠️ Note

Always run `terraform destroy` after testing to avoid AWS charges.
